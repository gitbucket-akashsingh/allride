# AllRide

AllRide is a full-stack ride-sharing platform inspired by Uber.

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Leaflet Maps

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL

## Features

- Authentication & Authorization
- Rider Dashboard
- Driver Dashboard
- Maps & Routing
- Real-time Ride Tracking
- WebSocket Integration
